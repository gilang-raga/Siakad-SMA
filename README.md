# Sistem Informasi Akademik (SIAKAD) 

## Deskripsi

Sistem Informasi Akademik (SIAKAD) berbasis web yang dirancang untuk membantu pengelolaan data akademik sekolah secara terintegrasi. Sistem menyediakan fitur manajemen data siswa, guru, autentikasi pengguna, dan dashboard berdasarkan role pengguna.

## Fitur Utama

* Login dan autentikasi pengguna
* Dashboard Admin
* Dashboard Guru
* Dashboard Siswa
* Manajemen Data Siswa
* Manajemen Data Guru
* Role-based Access Control
* Integrasi Email Notification

## Teknologi yang Digunakan

### Frontend

* React.js
* TypeScript
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Instalasi

```bash
npm install
npm run dev
```
